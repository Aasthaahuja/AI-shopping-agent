# Kasparro Fashion — AI Shopping Agent

An AI-powered shopping assistant built on Shopify for the Kasparro Agentic Commerce Hackathon.

## What it does
- Understands natural language shopping requests
- Recommends products with reasoning
- Filters by budget intelligently
- Adds items to cart and generates checkout link

## Tech Stack
- Python + Flask
- Groq API (Llama 3.3-70b)
- Shopify Storefront API (GraphQL)

## Setup
1. Clone the repo
2. Copy `.env.example` to `.env` and fill in your keys
3. `pip install -r requirements.txt`
4. `python app.py`
